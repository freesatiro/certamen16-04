# 🚀 Portafolio Profesional - Martin Seguel

Este proyecto es un portafolio personal desarrollado para el certamen de la asignatura de **Desarrollo Web Front-End**. El sitio es una Single Page Application (SPA) que utiliza estándares modernos de HTML5 y CSS3, con un enfoque en la semántica y el diseño responsivo.

## 📋 Características Técnicas

* **HTML5 Semántico:** Uso estricto de etiquetas como `<header>`, `<nav>`, `<main>`, `<article>`, `<figure>`, `<aside>` y `<footer>` para mejorar el SEO y la accesibilidad.
* **CSS3 Moderno:** * Implementación de **Flexbox** para la barra de navegación y alineación de elementos.
    * Uso de **CSS Grid** para la galería de proyectos y la sección de contacto.
    * Diseño **Responsivo** que se adapta a dispositivos móviles y escritorio.
* **Interactividad:** Formulario de contacto funcional (UI) con validaciones nativas de HTML5.
* **Tipografía:** Integración de fuentes de Google (Google Sans) para una estética limpia.

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* Git & GitHub
